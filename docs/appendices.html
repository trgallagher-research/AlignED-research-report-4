Appendix A: Full prompts

Condition A: Unprompted

You are an experienced mathematics teacher. Create a set of 4 worked examples teaching Year 8 students how to solve two-step linear equations (e.g., 3x + 5 = 20). Each worked example should present a problem and its full worked solution.

Condition B: General CLT

You are an experienced mathematics teacher. Create a set of 4 worked examples teaching Year 8 students how to solve two-step linear equations (e.g., 3x + 5 = 20). Each worked example should present a problem and its full worked solution. Apply cognitive load theory principles in your design.

Condition C: Specific fading

You are an experienced mathematics teacher. Create a set of 4 worked examples teaching Year 8 students how to solve two-step linear equations (e.g., 3x + 5 = 20). Apply the worked example fading effect from cognitive load theory: systematically remove solution steps across the sequence so that learners gradually take over more of the problem-solving process.

Design notes: Conditions A and B use "set" rather than "sequence" to avoid priming CLT associations ("worked example sequence" is a CLT term of art). Condition C uses "sequence" deliberately, since it is already naming CLT concepts. Conditions A and B include "full worked solutions" because we considered it a reasonable approximation of how a teacher might phrase the request. Condition C removes that phrasing and asks for fading directly. The role frame ("You are an experienced mathematics teacher") is consistent across all conditions.

Appendix B: Full scoring rubric

Dimension 1: Output structure (0–2)

ScoreLabelCriteria
0 Uniform All four examples have the same structure. Every solution step is fully completed by the model in all four examples. The examples may vary in difficulty but not in how much work the student does.
1 Some variation There is some structural variation across examples (shorter explanations, different formatting). All solution steps are still completed by the model. No steps are left for the student to complete.
2 Progressive scaffolding removal Solution steps are progressively removed across the set. At least one example is partially completed (some steps left blank, marked with "?", or explicitly left for the student), and the final example requires the student to complete most or all steps independently.

Key distinctions:

  • Harder equations with all steps shown = 0 or 1, not scaffolding removal.
  • Less narration but all steps present = 1, not scaffolding removal.
  • Steps left for the student to complete = 2.

Dimension 2: Reasoning trace content (0–2)

ScoreLabelCriteria
0 Absent Trace does not reference fading, scaffolding removal, or the logic of gradually shifting work to the learner.
1 Implicit logic Trace references the underlying logic without CLT terminology. Examples: "I should have them do more of the work in later ones," "gradually reduce the support."
2 Explicit reference Trace explicitly names: fading, the worked example effect, the completion problem effect, cognitive load theory, or Sweller.

Gemini 3.1 Pro — Condition A (Unprompted) 
**My Plan for Crafting Two-Step Equation Examples**

Right, let's get down to it. I need to create four worked examples for Year 8 students on solving two-step linear equations. My aim is to make it crystal clear, encouraging, and step-by-step, targeting those crucial 12-14-year-olds.

First, I've got to ensure the pedagogical approach is spot on. For this age group, the balance method is the way to go—doing the same thing to both sides is intuitive and powerful. I'll make sure to explicitly state the core goal: to isolate that variable, getting 'x' (or whatever letter we're using) all by itself. I'll highlight the reverse order of operations – undoing addition/subtraction *before* multiplication/division.

To cover a good range of examples, I'll vary the equations. Example one will be a straightforward addition and multiplication problem like *3x + 5 = 26*.  The second example will use subtraction, such as *4y - 7 = 13*. Example three will include division: *m/2 + 4 = 10*.

Finally, to challenge them slightly and reinforce the understanding that equations work both ways, Example four will involve a negative coefficient or the variable on the right side. I thought about a few: maybe a negative coefficient example *15 - 2k = 3* or even an equation with the variable on the right like *22 = 5p - 3* and decided that the negative coefficient example is the perfect way to solidifying the basics.

With these examples planned, I'll start drafting.  I'll begin each with a friendly introduction.  Each worked solution will be meticulously broken down:

*   **Example 1:** Solve *3x + 5 = 26* (subtract 5, then divide by 3)
*   **Example 2:** Solve *4y - 7 = 13* (add 7, then divide by 4)
*   **Example 3:** Solve *m/2 + 4 = 10* (subtract 4, then multiply by 2).
*   **Example 4:** Solve *15 - 2k = 3* (subtract 15, then divide by -2) or $22 = 5p - 3$. I will add 3, then divide by 5.

I'll include a 'check' step in the first example to model good practice.

Once the examples are drafted, I'll review them with a Year 8 lens. Is the language right? Are the steps clear and well-separated?  I'll use the proper terminology: 'variable', 'inverse operation', 'isolate'. I think I'll also add a little "Teacher Tip" to each one or maybe a general one at the beginning, just to give a little extra guidance.

Finally, I'll tidy up the presentation with clear markdown formatting, using bolding and bullet points to make it easily readable, whether it's for a worksheet or whiteboard notes.
Gemini 3.1 Pro — Condition B (General CLT) 
**My Thought Process for Designing Two-Step Equation Examples**

Right, let's craft some effective worked examples for Year 8 students on solving two-step linear equations. My aim, as an experienced teacher, is to make these examples not just *solutions*, but solid learning experiences. I need to keep Cognitive Load Theory (CLT) at the forefront of my mind.

First, I need to understand the request. It's a series of worked examples, solving equations like $3x + 5 = 20$. The target is Year 8 students, so I must start simple and use clear language. The format needs to be concise, step-by-step and should follow all CLT principles.

To achieve this, I'll leverage CLT principles specifically: I'll lean heavily on the Worked Example Effect. This means providing *complete* solutions with annotations, like an expert teacher would do. This helps reduce their cognitive load. I'll also incorporate "fading" by starting with simpler equations and gradually increasing complexity, but maintaining a similar format. Split-attention is the enemy, so the explanations will be right alongside the math, not separated. I'll use bolding and clear layout to guide their attention, signaling key steps, and I'll keep the structure of each example *consistent*, so they learn the process, not just the format.

I'll arrange the examples in increasing difficulty. Example 1 will be basic, positive integers, like $3x + 4 = 19$. Example 2 will introduce subtraction, still with positive integers (e.g., $4y - 7 = 13$). Example 3 will involve division, like $\frac{m}{2} + 5 = 11$, and finally, Example 4 will have either a negative outcome or a negative coefficient. I decided on a negative result to keep it accessible for Year 8 (e.g., $5p + 12 = 2$). I want to provide variety in the types of operations needed to show how the "inverse operation" concept applies regardless of the numbers involved.

For drafting the examples, I'll use a clear, side-by-side format: the equation on one side, and a *brief* explanation or "Think Aloud" prompt next to it. For example 1, I'll start with "Solve $3x + 4 = 19$. Goal: Get *x* by itself." Then, I'll show Step 1, "Subtract 4: $3x = 15$", and Step 2, "Divide by 3: $x = 5$". I'll include a *quick* self-check to encourage that vital habit: "$3(5) + 4 = 19$. Good!" The other examples follow a similar structure.

The layout is key. I'll use a table structure (Markdown works great) to keep the explanations directly beside the math. I'll also start with a "Teacher's Note" to introduce the overarching concept: "Reverse BIDMAS/BODMAS" and the rule of equality: "Whatever you do to one side, you must do to the other." The language will be simple and I'll consistently use "Inverse operation."

I'll then review the draft against my constraints and intentions: experienced teacher tone – check, 4 worked examples – check, Year 8 level – perfectly aligned, and CLT principles applied – absolutely. Finally, I'll add an introduction and conclusion to set the context and ensure the markdown formatting is clean. It's about setting them up for success, so, let's keep it focussed and relevant.
Gemini 3.1 Pro — Condition C (Specific fading) 
**My Thought Process: Designing a Worked Example Fading Sequence for Year 8 Algebra**

Okay, so I'm putting together a worked example sequence for my Year 8s on solving two-step linear equations. We're using the worked example fading effect here, also known as backward fading, and this is based on cognitive load theory. I want to give them four examples to work through, gradually increasing their responsibility for the solution.

First, I need to make sure I understand the core steps: identifying which operation to undo first (addition/subtraction), using inverse operations to isolate the variable, and checking the answer.

I'm starting with a fully worked example ($4x + 7 = 23$). I'll show every step, explaining each one clearly. Next, for the second example ($5y - 3 = 22$), the students will do the very last step. I'll scaffold them by telling them to add 3 to both sides.

For the third example ($2m + 9 = 21$), I'll fade out the last two steps. But I need to adjust this based on the *backward fading* principal. I'll provide the first step for them, then have them complete the execution. I need to make sure they're clear on how to isolate the variable. This will structure it like this:

*   Eq: $2m + 9 = 21$
*   Step 1: Subtract 9 from both sides. -> $2m = \_\_\_$ (Student fills in 12)
*   Step 2: ____________ (Student writes: Divide both sides by 2)
*   Final Answer: $m = \_\_\_$ (Student fills in 6)

The fourth example ($6k - 5 = 19$) is the independent practice. I want to give them a clear prompt to use what they've learned, and I'll see how they go.

The numbers should be friendly (integer solutions only). I'll use bolding, bullet points, and blank spaces to make it visually accessible.

The fading should be: Example 1, 100% teacher. Example 2, 75% teacher, 25% student. Example 3, 25% teacher, 75% student. And Example 4: 0% teacher, 100% student. The worked example fading principle is adhered to.

I'll write an introduction to set the context. I'll also add a brief note on the importance of checking answers because that's good practice. I'll make sure the layout clearly indicates where the student is supposed to write using underscore lines `_______`. I have to be mathematically accurate (using terms like "inverse operations").
