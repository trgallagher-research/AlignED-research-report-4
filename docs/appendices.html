Appendix A: Full prompts

Condition A: Baseline

You are an experienced mathematics teacher. Create a set of 4 worked examples teaching Year 8 students how to solve two-step linear equations (e.g., 3x + 5 = 20). Each worked example should present a problem and its full worked solution.

Condition B: General CLT

You are an experienced mathematics teacher. Create a set of 4 worked examples teaching Year 8 students how to solve two-step linear equations (e.g., 3x + 5 = 20). Each worked example should present a problem and its full worked solution. Apply cognitive load theory principles in your design.

Condition C: Specific fading

You are an experienced mathematics teacher. Create a set of 4 worked examples teaching Year 8 students how to solve two-step linear equations (e.g., 3x + 5 = 20). Apply the worked example fading effect from cognitive load theory: systematically remove solution steps across the sequence so that learners gradually take over more of the problem-solving process.

Design notes: Conditions A and B use "set" rather than "sequence" to avoid priming CLT associations ("worked example sequence" is a CLT term of art). Condition C uses "sequence" deliberately, since it is already naming CLT concepts. Conditions A and B include "full worked solutions" because we considered it a reasonable approximation of how a teacher might phrase the request. Condition C removes that phrasing and asks for fading directly. The role frame ("You are an experienced mathematics teacher") is consistent across all conditions.

Appendix B: Full scoring rubric

Dimension 1: Output structure (0–2)

ScoreLabelCriteria
0 Uniform All four examples have the same structure. Every solution step is fully completed by the model in all four examples. The examples may vary in difficulty but not in how much work the student does.
1 Some variation There is some structural variation across examples (shorter explanations, different formatting). All solution steps are still completed by the model. No steps are left for the student to complete.
2 Progressive scaffolding removal Solution steps are progressively removed across the set. At least one example is partially completed (some steps left blank, marked with "?", or explicitly left for the student), and the final example requires the student to complete most or all steps independently.

Key distinctions:

  • Harder equations with all steps shown = 0 or 1, not scaffolding removal.
  • Less narration but all steps present = 1, not scaffolding removal.
  • Steps left for the student to complete = 2.

Dimension 2: Reasoning trace content (0–2)

ScoreLabelCriteria
0 Absent Trace does not reference fading, scaffolding removal, or the logic of gradually shifting work to the learner.
1 Implicit logic Trace references the underlying logic without CLT terminology. Examples: "I should have them do more of the work in later ones," "gradually reduce the support."
2 Explicit reference Trace explicitly names: fading, the worked example effect, the completion problem effect, cognitive load theory, or Sweller.

## Example 4
\[

### Solution

When the variable appears on **both sides**, first bring all the \(x\)-terms to one side and the constants to the other.

|------|------------|-----|

|   | -> \(x = -11\) | **Answer** |

### Quick Checklist for Solving Two-Step Equations

1. **Identify** whether the constant is added/subtracted and whether the variable is multiplied/divided.
2. **Undo the addition/subtraction** first (move the constant to the opposite side).
3. **Undo the multiplication/division** next (divide or multiply by the coefficient of the variable).
4. **Check**: Substitute your answer back into the original equation to verify it works.

Practice these steps, and the two-step linear equations will become routine!

## Solving Two-Step Linear Equations -- Worked Examples
*Target: Year 8 (Key Stage 3) -- one variable, two-step equations*

Below are four complete worked examples. Each example follows the same **four-step structure** so that you can see exactly what to do, why you do it, and where to write each step. The layout is deliberately simple (few symbols on a line, colour-coded boxes) to keep the **cognitive load** low while you build a strong procedural habit.

---

### General 4-Step Template

| Step | What you do | Why it works |
|------|-------------|--------------|
| **1. Identify the "extra" number** | Look for the term that is **added** or **subtracted** from the variable term (e.g., `+5` or `--3`). | This term does not belong to the variable part; we must remove it first. |
| **2. Remove the extra number** | Use the opposite operation on **both sides** of the equation (subtract if it was added, add if it was subtracted). | Keeping the equation balanced preserves equality. |
| **3. Isolate the variable coefficient** | Now you have something like `ax = b`. Divide (or multiply) both sides by `a` to get `x = b/a`. | This gives the value of the unknown. |
| **4. Check** | Substitute the answer back into the original equation. | Confirms the solution and builds confidence. |

## Example 1
**Problem:** `3x + 5 = 20`

| Step | Work (write exactly as shown) | Explanation |
|------|------------------------------|-------------|
| 1 | **Identify the extra number** -> `+5` is added to the `3x` term. | We must get rid of the `+5` first. |
| 2 | **Subtract 5 from both sides**  `3x + 5 - 5 = 20 - 5`  `3x = 15` | Subtracting the same number from both sides keeps the equality true. |
| 3 | **Divide by the coefficient of x** `3x / 3 = 15 / 3` `x = 5` | Division isolates `x`. |
| 4 | **Check** Replace `x` by `5` in the original: `3*5 + 5 = 15 + 5 = 20` | The left-hand side equals the right-hand side, so `x = 5` is correct. |

## Example 2
**Problem:** `-2x - 7 = 1`

| Step | Work | Explanation |
|------|------|-------------|
| 1 | **Extra number** -> `-7` (it is subtracted). | We need to "undo" the subtraction. |
| 2 | **Add 7 to both sides** `-2x - 7 + 7 = 1 + 7` `-2x = 8` | Adding the opposite (+7) removes the `-7`. |
| 3 | **Divide by the coefficient** (-2) `-2x / (-2) = 8 / (-2)` `x = -4` | Division gives the value of `x`. |
| 4 | **Check** `-2(-4) - 7 = 8 - 7 = 1` | Works, so `x = -4`. |

| 3 | **Divide by 4** `4x / 4 = 12 / 4` `x = 3` | Isolate `x`. |

The "worked-example fading" technique means we start with a **complete** solution and then **remove** steps one by one. By the time you reach the fourth example you will be doing most of the work yourself.

## Example 1 -- **Full solution shown**

Solve **\(3x+5 = 20\)**

| Step | What we do | Why we do it |
|------|------------|--------------|

| 2  | Simplify: \(3x = 15\) | The equation is now **one-step** (only a multiplication). |
| 3  | Divide both sides by 3: \(\displaystyle x =\frac{15}{3}\) | To isolate the variable. |
| 4  | Simplify: \(x = 5\) | Final answer. |

**Check**: \(3(5)+5 = 15+5 = 20\)

### How to use the fading sequence

1. **Study Example 1** -- follow every algebraic move.
2. **Do Example 2** -- fill in the hidden simplification (just one arithmetic step).
3. **Do Example 3** -- fill in two missing steps; this forces you to decide which operation comes first.
4. **Do Example 4** completely on your own, using the pattern you have learned.

1. **Undo the constant term** (add or subtract).
